## 二、短线动量策略要点

### 什么是短线动量策略？
基于"强者恒强"假设：近期涨幅大的股票未来一段时间倾向于继续跑赢大盘，持仓周期通常1-5天。

### 核心逻辑
1. 趋势确认：价格突破均线或前期高点
2. 动量验证：RSI/MACD等指标配合确认趋势强度
3. 动量衰竭识别：提前发现趋势反转信号

### 常用技术指标
| 指标 | 核心用法 |
|------|----------|
| RSI | 超买区>70，超卖区<30 |
| MACD | DIF上穿DEA=买入，死叉=卖出 |
| 布林带 | 价格触及上轨可能回调，触及下轨可能反弹 |
| 成交量 | 放量上涨=确认趋势，缩量上涨=预警反转 |
| KDJ | 金叉/死叉配合超买超卖区间 |

### 动量衰竭识别
1. **价格创新高 + RSI背离**（价格创新高但RSI低于前高）
2. **量价背离**：成交量萎缩但价格继续上涨
3. 连续长上影线/流星线
4. 均线开始走平无趋势方向
5. MACD红柱面积缩减

## 四、策略风险提示

### 常见陷阱
1. **动量反转陷阱**：趋势高潮时追入，正好买在最高点
2. **市场风格切换风险**：动量策略在趋势市场有效，在震荡市失效
3. **时间衰减风险**：动量效应短期内最强，持仓越久超额收益越低
4. **交易成本侵蚀**：频繁交易的手续费和滑点会显著影响收益
5. **幸存者偏差**：看到涨停股忽视大量"失败"的动量股

### 当前市场环境风险
1. 近期市场波动较大，需控制仓位
2. 紧盯成交量变化，缩量上涨需警惕
3. 避免追入连续3板以上的个股
4. 关注外围市场（美股、港股）联动效应
5. 节假日前夕流动性下降，谨慎操作
