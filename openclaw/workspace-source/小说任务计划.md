# 小说检查与优化任务计划

## 任务目标
检查并优化《末世摆烂：我白给，她....》以下章节段：
- 第52-100章（49章）
- 第122-153章（32章）
- 第303-310章（8章）
- 第320-408章（89章）
- 第409-581章（173章）
- 第583-609章（27章）

## 任务拆解

### 阶段一：了解现状
- [ ] 1. 检查大纲文件是否存在
- [ ] 2. 统计各章节段现有字数
- [ ] 3. 确认每章是否存在

### 阶段二：创建/完善大纲
- [ ] 4. 分析现有剧情，创建四卷大纲
- [ ] 5. 确定每卷主题和字数目标

### 阶段三：分卷检查与优化（每卷独立）
**第1卷检查**（52-100章）
- [ ] 检查是否符合大纲主题
- [ ] 检查字数要求
- [ ] 检查情节通顺度
- [ ] 检查上下文衔接
- [ ] 文笔优化润色
- [ ] 封装第1卷

**第2卷检查**（122-153章）
- [ ] 同上流程

**第3卷检查**（303-310, 320-408章）
- [ ] 同上流程

**第4卷检查**（409-581, 583-609章）
- [ ] 同上流程

### 阶段四：主封装
- [ ] 所有卷检查完成后进行主封装

## 字数要求（需确认）
每章目标字数：约2000-3000字

## 注意事项
- 长任务拆分为短任务
- 每卷自检完成后才能进入下一卷
- 先润色优化再封装
