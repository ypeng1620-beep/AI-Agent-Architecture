# Development API 集合

## 概述
本技能收录20个免费的Development相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥
- **OAuth**: 需要OAuth认证

## 常用API快速调用
### JSONStore
**认证**: No
**CORS**: Yes
```bash
curl -s "https://www.jsonstore.io/"
```

### Random User
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://randomuser.me/"
```

### CountAPI
**认证**: No
**CORS**: Yes
```bash
curl -s "https://api.countapi.xyz/"
```

### Country
**认证**: No
**CORS**: Yes
```bash
curl -s "http://country.is/"
```

### IP
**认证**: No
**CORS**: Yes
```bash
curl -s "https://ip.horse/"
```
