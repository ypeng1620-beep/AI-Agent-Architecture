# Government API集合

## 概述
收录18个Government相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请

## 常用API
### Brazil
```bash
curl -s "https://brasilapi.com.br/"
```
### City Toronto
```bash
curl -s "https://open.toronto.ca/"
```
### Open Gov Norway
```bash
curl -s "https://data.norge.no/dataservices"
```
### Open Gov Poland
```bash
curl -s "https://dane.gov.pl/en"
```
### Open Gov Portugal
```bash
curl -s "https://dados.gov.pt/en/docapi/"
```
