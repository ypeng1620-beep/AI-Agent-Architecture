# Cryptocurrency API 集合

## 概述
本技能收录14个免费的Cryptocurrency相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥

## 常用API快速调用
### CoinGecko
**认证**: No
**CORS**: Yes
```bash
curl -s "https://www.coingecko.com/en/api"
```
