# Games & Comics API集合

## 概述
收录20个Games & Comics相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### Pokéapi
```bash
curl -s "https://pokeapi.co"
```
### JokeAPI
```bash
curl -s "https://sv443.net/jokeapi/v2/"
```
### Rick and Morty
```bash
curl -s "https://rickandmortyapi.com"
```
### xkcd
```bash
curl -s "https://xkcd.com/json.html"
```
### Dungeons and Dragons Alt
```bash
curl -s "https://open5e.com/"
```
