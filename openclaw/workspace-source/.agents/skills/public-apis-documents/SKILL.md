# Documents & Productivity API 集合

## 概述
本技能收录20个免费的Documents & Productivity相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥
- **OAuth**: 需要OAuth认证

## 常用API快速调用
### Vector Express
**认证**: No
**CORS**: No
```bash
curl -s "https://vector.express"
```

### WakaTime
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://wakatime.com/developers"
```
