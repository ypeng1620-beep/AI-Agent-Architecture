# Cloud Storage & File Sharing API 集合

## 概述
本技能收录23个免费的Cloud Storage & File Sharing相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥
- **OAuth**: 需要OAuth认证

## 常用API快速调用
### Pantry
**认证**: No
**CORS**: Yes
```bash
curl -s "https://getpantry.cloud/"
```

### File.io
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://www.file.io"
```

### AnonFiles
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://anonfiles.com/docs/api"
```

### BayFiles
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://bayfiles.com/docs/api"
```

### Surge
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://surge.sh/"
```
