# Open Source Projects API集合

## 概述
收录9个Open Source Projects相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **OAuth**: 需要认证

## 常用API
### GitHub ReadMe Stats
```bash
curl -s "https://github.com/anuraghazra/github-readme-stats"
```
### GitHub Contribution Chart
```bash
curl -s "https://github-contributions.vercel.app"
```
### Evil Insult Generator
```bash
curl -s "https://evilinsult.com/api"
```
### Shields
```bash
curl -s "https://shields.io/"
```
### Datamuse
```bash
curl -s "https://www.datamuse.com/api/"
```
