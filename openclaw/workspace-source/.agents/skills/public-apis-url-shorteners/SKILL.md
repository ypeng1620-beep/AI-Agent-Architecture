# URL Shorteners API集合

## 概述
收录11个URL Shorteners相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### Shrtcode
```bash
curl -s "https://shrtco.de/docs"
```
### CleanURI
```bash
curl -s "https://cleanuri.com/docs"
```
### GoTiny
```bash
curl -s "https://github.com/robvanbakel/gotiny-api"
```
### 1pt
```bash
curl -s "https://github.com/1pt-co/api/blob/main/README.md"
```
### Git.io
```bash
curl -s "https://github.blog/2011-11-10-git-io-github-url-shortener/"
```
