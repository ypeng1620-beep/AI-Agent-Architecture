# Sports & Fitness API集合

## 概述
收录18个Sports & Fitness相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### balldontlie
```bash
curl -s "https://www.balldontlie.io"
```
### City Bikes
```bash
curl -s "https://api.citybik.es/v2/"
```
### Football Standings
```bash
curl -s "https://github.com/azharimm/football-standings-api"
```
### OpenLigaDB
```bash
curl -s "https://www.openligadb.de"
```
### Sport List Data
```bash
curl -s "https://developers.decathlon.com/products/sports"
```
