# Social API集合

## 概述
收录30个Social相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### HackerNews
```bash
curl -s "https://github.com/HackerNews/API"
```
### 4chan
```bash
curl -s "https://github.com/4chan/4chan-API"
```
### Lanyard
```bash
curl -s "https://github.com/Phineas/lanyard"
```
### Hashnode
```bash
curl -s "https://hashnode.com"
```
### Open Collective
```bash
curl -s "https://docs.opencollective.com/help/developers/api"
```
