# Machine Learning API集合

## 概述
收录16个Machine Learning相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### OpenVisionAPI
```bash
curl -s "https://openvisionapi.com"
```
### Deepcode
```bash
curl -s "https://www.deepcode.ai"
```
### EXUDE-API
```bash
curl -s "http://uttesh.com/exude-api/"
```
