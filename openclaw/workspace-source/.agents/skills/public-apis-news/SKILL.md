# News API集合

## 概述
收录15个News相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### Spaceflight News
```bash
curl -s "https://spaceflightnewsapi.net"
```
### Inshorts News
```bash
curl -s "https://github.com/cyberboysumanjay/Inshorts-News-API"
```
