# Currency Exchange API 集合

## 概述
本技能收录16个免费的Currency Exchange相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥

## 常用API快速调用
### Frankfurter
**认证**: No
**CORS**: Yes
```bash
curl -s "https://www.frankfurter.app/"
```

### Exchangerate.host
**认证**: No
**CORS**: Yes
```bash
curl -s "https://exchangerate.host/"
```

### National Bank of Poland
**认证**: No
**CORS**: Yes
```bash
curl -s "http://api.nbp.pl/"
```

### Russian Central Bank
**认证**: No
**CORS**: Yes
```bash
curl -s "https://cbr.ru/eng/currency_base/"
```

### VATComply
**认证**: No
**CORS**: Yes
```bash
curl -s "https://www.vatcomply.com/"
```
