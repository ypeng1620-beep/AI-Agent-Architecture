# Food & Drink API集合

## 概述
收录24个Food & Drink相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### Open Brewery DB
```bash
curl -s "https://www.openbrewerydb.org"
```
### Foodish
```bash
curl -s "https://github.com/surhud004/Foodish#readme"
```
### BaconMockup
```bash
curl -s "https://baconmockup.com/"
```
### Coffee
```bash
curl -s "https://coffee.alexflipnote.dev/"
```
### Fruityvice
```bash
curl -s "https://www.fruityvice.com"
```
