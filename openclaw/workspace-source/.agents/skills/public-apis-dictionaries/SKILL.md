# Dictionaries API 集合

## 概述
本技能收录21个免费的Dictionaries相关API。

## 认证说明
- **无需认证**: 可以直接调用
- **API Key**: 需要申请API密钥

## 常用API快速调用
### DictionaryAPI
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://dictionaryapi.net/"
```

### Wiktionary
**认证**: No
**CORS**: Yes
```bash
curl -s "https://en.wiktionary.org/w/api.php"
```

### DataMuse
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://www.datamuse.com/api/"
```

### OneLook
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://www.onelook.com/"
```

### Free
**认证**: No
**CORS**: Unknown
```bash
curl -s "https://www.free.wtf/"
```
