# Email API集合

## 概述
收录16个Email相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请

## 常用API
### Disify
```bash
curl -s "https://www.disify.com/"
```
### Eva
```bash
curl -s "https://eva.pingutil.com/"
```
### Kickbox
```bash
curl -s "https://open.kickbox.com/"
```
### Guerrilla Mail
```bash
curl -s "https://www.guerrillamail.com/GuerrillaMailAPI.html"
```
### mail.gw
```bash
curl -s "https://docs.mail.gw"
```
