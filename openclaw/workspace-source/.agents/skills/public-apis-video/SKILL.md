# Video API集合

## 概述
收录25个Video相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### An API of Ice And Fire
```bash
curl -s "https://anapioficeandfire.com/"
```
### Breaking Bad
```bash
curl -s "https://breakingbadapi.com/documentation"
```
### Breaking Bad Quotes
```bash
curl -s "https://github.com/shevabam/breaking-bad-quotes"
```
### Final Space
```bash
curl -s "https://finalspaceapi.com/docs/"
```
### Harry Potter Characters
```bash
curl -s "https://hp-api.herokuapp.com/"
```
