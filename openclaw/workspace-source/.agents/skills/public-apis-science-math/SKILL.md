# Science & Math API集合

## 概述
收录29个Science & Math相关免费API。

## 认证说明
- **无需认证**: 可直接调用
- **API Key**: 需要申请
- **OAuth**: 需要认证

## 常用API
### arXiv
```bash
curl -s "https://arxiv.org/help/api/user-manual"
```
### NASA
```bash
curl -s "https://api.nasa.gov"
```
### Launch Library 2
```bash
curl -s "https://thespacedevs.com/llapi"
```
### SpaceX
```bash
curl -s "https://github.com/r-spacex/SpaceX-API"
```
### SpaceX GraphQL
```bash
curl -s "https://api.spacex.land/graphql/"
```
