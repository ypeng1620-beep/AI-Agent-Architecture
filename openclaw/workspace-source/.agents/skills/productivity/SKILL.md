# 中国大陆独立站收款方案

## 核心问题

Gumroad、Stripe 等国际平台默认不支持微信/支付宝，需要先解决支付方式问题。

## 推荐方案

### 爱发电 (afdian.net) — 最快

- **优点**：个人可开，支持微信/支付宝，扫码支付自动发货，操作简单
- **适用**：虚拟产品、预售、会员、数字工具
- **缺点**：平台抽成（约5%）
- **注册**：afdian.net

### 即刻

- **优点**：创作者生态，有赞赏/预售功能
- **适用**：创作者社区、粉丝经济

### Paddle

- **优点**：支持微信支付，国际平台
- **缺点**：注册复杂，需要企业资质

## 流程

1. 注册爱发电账号（个人身份证即可）
2. 创建产品，设置价格
3. 获取产品链接
4. 替换独立站购买按钮的链接

## 独立站配置修改

在 `src/config.ts` 中修改：

```ts
gumroadLink: "https://afdian.net/item/你的产品ID"
```

同时更新按钮文案为"立即支持"或"购买"。

## 注意事项

- 预售产品：在爱发电产品描述写清楚发货时间
- 退款政策：在爱发电设置里配置
- 微信/支付宝买家体验：扫码 → 支付 → 自动收到下载链接，全程无需额外操作
