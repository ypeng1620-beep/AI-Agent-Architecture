# NovelMaster Skill - 设计规范

## 1. 项目概述

**项目名称**: NovelMaster  
**类型**: OpenClaw AI 小说写作技能  
**核心理念**: 集百家之长，AI 生成 → 多层审核 → 人类把关 → 迭代优化

## 2. 功能设计

### 核心功能模块

#### 2.1 创作规划
- **雪花写作法**: 提炼一句话核心 → 一页大纲 → 人物素描 → 四页大纲 → 详细大纲
- **大纲生成**: 基于题材自动生成结构化大纲
- **世界观构建**: 自动生成时代背景、势力设定

#### 2.2 内容生成
- **章节生成**: 大纲→章节正文的完整流程
- **情节续写**: 基于已有内容智能续写
- **文风转换**: 保持风格一致性的内容生成

#### 2.3 质量审核 (多层)
- **AI 自审**: 语法、逻辑、一致性检查
- **伏笔审核**: 检测伏笔埋设与回收
- **期待感审核**: 情节张力评估
- **AI 检测规避**: humanizer 风格处理

#### 2.4 人类把关
- **分段审核**: 每章提交人类确认
- **修改反馈**: 支持人类修改指令
- **版本管理**: 历史版本可回溯

#### 2.5 输出格式
- **多格式导出**: Markdown、TXT、EPUB、PDF
- **统计报告**: 字数统计、章节进度、伏笔清单

## 3. 技术方案

- **架构**: 模块化技能系统
- **数据**: JSON 文件存储，无外部依赖
- **触发**: 关键词自动识别

## 4. 验收标准

- [ ] 支持雪花写作法流程
- [ ] 章节生成功能正常
- [ ] 多层审核机制可用
- [ ] 人类把关流程完整
- [ ] 导出功能可用
- [ ] 单元测试通过
